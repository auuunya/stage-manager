## 8. 关键决策 (ADR)

> **决策点**：令牌存储方案从 Redis 换回 JWT Stateless

- **背景**：初期考虑使用 Redis 存储 Session 以便于强制下线，但考虑到首期需要快速横向扩展且服务器内存受限。
- **对比方案**：Redis-Session (服务端有状态) vs JWT (客户端无状态)。
- **结论**：采用 JWT，并将权限信息加密在 Payload 中。
- **影响**：前端需要在每次请求头带上 Authorization，且无法在服务端主动撤销 Token（需等待过期）。
